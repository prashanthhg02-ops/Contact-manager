# Contact-manager
1. Contact Manager

A Contact Manager is a software application used to store, organize, search, update, and manage contact information such as names, phone numbers, email addresses, and addresses.

2. HTML (HyperText Markup Language)

HTML is the standard markup language used to create the structure and content of web pages.

3. CSS (Cascading Style Sheets)

CSS is a stylesheet language used to design and style web pages by controlling colors, fonts, layouts, and responsiveness.

4. JavaScript

JavaScript is a programming language used to make web pages interactive by handling user actions, form validation, and dynamic content.

5. Python

Python is a high-level, easy-to-learn programming language used for developing web applications, automation, data analysis, and more.

6. Flask

Flask is a lightweight Python web framework used to build web applications and create APIs quickly and efficiently.

7. SQLite

SQLite is a lightweight, serverless database that stores data in a single file and is commonly used for small to medium-sized applications.

8. Database

A database is an organized collection of data that allows information to be stored, retrieved, updated, and deleted efficiently.

9. CRUD Operations

CRUD stands for Create, Read, Update, and Delete, which are the four basic operations performed on data in a database.

10. Frontend

The frontend is the part of the application that users interact with, including the interface designed using HTML, CSS, and JavaScript.

11. Backend

The backend is the server-side part of the application that processes requests, performs business logic, and communicates with the database.

12. User Interface (UI)

The User Interface (UI) is the visual layout of an application that allows users to interact with its features easily.

13. Responsive Design

Responsive Design is a web design approach that ensures a website adapts and displays correctly on desktops, tablets, and mobile devices.

14. Contact Form

A Contact Form is an input form used to collect contact details such as name, phone number, email, and address from the user.

15. Search Function

The Search Function enables users to quickly find a contact by entering a name, phone number, or email.

16. Data Validation

Data Validation is the process of checking user input to ensure that the information entered is correct, complete, and in the proper format.

17. API (Application Programming Interface)

An API is a set of rules that allows different software applications to communicate and exchange data with each other.

18. Contact Management System

A Contact Management System is an application that helps users efficiently manage personal or business contacts by providing features such as adding, editing, deleting, and searching contact records.
